The Information Card Icon

The Information Card icon is the standard visual cue to users that a site (or application) accepts an Information Card to perform an identity data transaction (register, login, form fill, share data, etc.). This icon is always displayed as clickable button that will launch a card selector provided the user has one installed (or if not, provide instructions about how to obtain one). The icon should never be used for any other purpose. In this respect it performs much the same function as the RSS icon that indicates the availability of an RSS feed.

For a non-clickable version of this icon, see the Information Card Outline Icon. For indicating ICF membership, see the Information Card Foundation Logos.

Guidelines for Usage

The ICF Website Best Practices Working Group is currently preparing guidelines for usage of the Information Card icon. They will posted as soon as they are ready.

Note that if you wish to use an icon to represent Information Card technology but NOT have it be mistaken for an active, clickable icon, please use the Information Card Outline Icon.

Download Options

The ICF is currently preparing a menu of download options for the Information Card icon. In the meantime you may download a ZIP file of the complete set of PNG images.
