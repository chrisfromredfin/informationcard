Following is a summary of the current active ICF Working Groups. See the Working Groups page of the ICF wiki for complete details.  (Those marked (IPR) require that each partipant first submit the ICF IPR Contribution Agreement.)

OASIS Coordination

Liaison with OASIS TCs working on standards related to Informatin Cards and the Identity Metasystem.

RP Evangelists

Develop materials and perform outreach to websites to accept Information Cards.

Website Best Practices (IPR)

Develop requirements, documentation, tutorials, and best practices for websites deploying Information Cards.

Schemas (IPR)

Promote interoperability of information cards through re-use of standard claim type URIs (Uniform Resource Identifiers).

Selector Interoperability (IPR)

Work on techical, policy, and user interface issues that directly affect interoperability of selectors, browsers, and servers.

Working Group participation is encouraged of all ICF members. If you are interested in starting a new Working Group:

  1. First check the Working Group wiki page listing all current and suggested working groups. See if you are able to work within one already formed. Email the Chairperson.
  2. If you believe a new one is needed, post a message to ICF All Members mailing list (see Mailing Lists) to propose the Working Group.
  3. If you are not an ICF Steering or Sponsor member, enlist one to propose formation of the Working Group.
  4. See specific instructions for more about forming a Working Group
